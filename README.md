# 📁 Dossier Images — Baay Mora Studio BMS

Créez un dossier nommé `images/` à côté des fichiers HTML et placez-y les images suivantes :

## Images requises

| Nom du fichier        | Usage                              | Dimensions recommandées |
|-----------------------|------------------------------------|-------------------------|
| `hero.jpg`            | Image plein écran page d'accueil   | 1920×1080 px minimum    |
| `about-hero.jpg`      | Bannière page À Propos             | 1920×800 px             |
| `photographer.jpg`    | Portrait du photographe            | 800×1066 px (3:4)       |
| `contact-photo.jpg`   | Image décorative page Contact      | 800×1066 px (3:4)       |
| `portfolio-1.jpg`     | Portfolio — Portrait studio        | 1200×900 px             |
| `portfolio-2.jpg`     | Portfolio — Mariage                | 900×1200 px             |
| `portfolio-3.jpg`     | Portfolio — Mode                   | 900×1200 px             |
| `portfolio-4.jpg`     | Portfolio — Corporate              | 1200×900 px             |
| `portfolio-5.jpg`     | Portfolio — Événement              | 1200×900 px             |
| `portfolio-6.jpg`     | Portfolio — Portrait artistique    | 1200×900 px             |

## ⚠️ Important

Remplacez **`221XXXXXXXXX`** dans tous les fichiers HTML par le vrai numéro WhatsApp du studio.

Exemple : `https://wa.me/221771234567?text=...`
